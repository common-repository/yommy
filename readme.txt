Integrates Yommy.com widget into posts and pages. 


== Installation ==

1 - unzip the file
2 - copy yommy folder under /wp-content/plugins
3 - login into wordpress admin panel, go to plugins section and activate Yommy plugin
4 - under settings menu, click Yommy menu and customize it
